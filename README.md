# 🛒 Retail Sales Analysis

An end-to-end Business Intelligence project analyzing retail sales data.

## 🛠️ Tools Used
- Python / Pandas — Data cleaning & analysis
- SQL (SQLite) — Business queries
- Power BI — Interactive dashboard

## 📁 Project Structure
- `data/` — Raw and cleaned datasets
- `notebooks/` — Python analysis notebooks
- `sql/` — SQL query files
- `visuals/` — Charts and screenshots
- `dashboard/` — Power BI dashboard file

## 🔍 Key Findings
- **Electronics** generated the highest revenue among all categories
- **25–34** age group is the biggest spending segment
- Sales peaked in **[your top month]** of 2023
